去年，Anthropic 執行長 Dario Amodei 在一篇部落格文章中預言，AI 產業即將產生的營收規模將讓今日科技巨頭相形見絀。當時懷疑者不少，如今數字說話了。

根據半導體與 AI 基礎設施研究機構 SemiAnalysis 廣泛流傳的報告，Anthropic 的年化營收跑率（ARR）已突破 **440 億美元**。這個數字在短短兩個月內從 3 月的 190 億美元翻倍，相較 2025 年底的 90 億美元更是暴增近五倍。換算下來，公司每天新增的年化營收約達 **9,600 萬美元**——幾乎是每小時燒出一棟台北豪宅的速度在創造收入。

## 史無前例的成長曲線

即便以 AI 時代慣常浮誇的標準衡量，這條成長曲線依然令人瞠目。2025 年 1 月，Anthropic 的 ARR 約為 10 億美元；2026 年 2 月達到 140 億；接著出現令人難以置信的加速：3 月 190 億、4 月 300 億（光這個數字當時就已讓媒體大幅報導，因為這代表 Anthropic 首度超越 OpenAI），5 月更衝破 440 億。

一位看過 Anthropic 內部數據的創投人士坦言：「我們研究過逾 200 家上市軟體公司的 IPO，從未見過這樣的成長速度。」值得注意的是，Anthropic 達到這個成績，據報所花費的模型訓練成本約只有 OpenAI 的四分之一。效率落差若能維持，意味著 Anthropic 可能在模型架構或市場策略上取得了結構性優勢，而非曇花一現。

## 意外的成長引擎：Claude Code

Anthropic 旗下的通用 Claude 模型雖已在各產業吸引企業客戶，但近期的爆發性成長，有一個更具體的原因：**Claude Code**——公司於 2025 年 5 月正式發布的自主式程式設計助理。

Claude Code 在正式上市後六個月內，年化營收已達 10 億美元，這樣的速度幾乎是 Anthropic 自稱比較基準中最快的企業軟體產品。此後加速未停。2026 年以來，Claude Code 的企業訂閱量成長四倍；企業端使用量目前佔 Claude Code 總營收超過一半，平均合約規模也持續擴大，顯示企業客戶正從試點轉向全面部署。

為何採用速度如此之快？Claude Code 能自主閱讀大型程式碼庫、跨檔案撰寫與除錯程式、執行測試並反覆迭代——這些工作過去不是需要資深工程師耗費大量時間，就是仰賴龐大的外包團隊。對規模化的工程組織而言，投資報酬的計算非常直觀，採用決策因此快速成形。

## 企業集中，而非消費者熱潮

Anthropic 營收結構中最關鍵的一個細節，是其客群組成。年花費 100 萬美元以上的企業客戶，貢獻了約 **80% 的總營收**。更重要的是，這個比例在公司整體數字爆炸性成長的同時仍維持穩定，顯示公司並非仰賴很快就會退燒的消費者嘗鮮消費。

客戶名單同樣令人印象深刻：全球財富百大企業中，70% 是 Claude 的客戶，包含市值前十大企業中的八家。年消費額達 100 萬美元以上的企業客戶數，在 Anthropic 完成 G 輪募資後短短不到兩個月內就從 500 家倍增至逾 1,000 家——募資公告本身似乎成了一種信任訊號，加速了企業採購決策。

為支撐這批客戶，Anthropic 已與 Google 和 Broadcom 擴大算力合作，預計 2027 年將有約 **35 億瓦的 Google TPU 算力**上線——這個數字充分說明，在這個規模下維持前沿模型性能，所需的資本密集程度有多高。

## OpenAI 的「帳務之爭」

Anthropic 4 月 300 億美元的 ARR 公告，引發了一場不尋常的公開反擊。OpenAI 首席營收長 Denise Dresser 據報向員工發出一份四頁備忘錄，主張 Anthropic 的數字被高估了約 80 億美元，原因在於兩家公司對雲端合作夥伴營收（包含超大規模雲端業者的算力承諾與積分）的認列方式不同。

這場爭議在兩家公司遞交 IPO 說明書、採用標準化會計準則之前，恐怕難有定論。但即便採用 Dresser 的修正數字，Anthropic 的調整後 ARR 仍將超越 OpenAI 的約 250 億美元。更重要的是，以目前的成長速度，這份備忘錄寫就以來，兩者之間的差距已進一步擴大。

## 對整個 AI 產業意味著什麼

Anthropic 的營收弧線，不只是一家公司的商業成功。它標誌著企業軟體採購模式正在被即時重寫。

長期以來，企業軟體的成長被認為受限於銷售週期、整合難度、資安審查與組織變革——這套阻力系統讓即使再好的產品也難以在大型組織中快速擴張。AI 代理人（Agents）顯然是個例外。投資報酬的論證足夠直接，部署模式也足夠靈活，讓原本需要 12 至 18 個月的採購週期，如今可能在數週內完成。

這種速度創造了複利式的先行優勢。Anthropic 在財富百大企業中不斷擴大的份額，意味著它正在累積專有的工作流程數據、回饋迴路與整合深度——即便競爭對手日後推出技術上更優越的模型，這些壁壘也將愈來愈難被取代。

2026 年初，AI 產業還在辯論前沿模型開發是否是一門可持續的生意。Anthropic 的 440 億美元年化營收，或許已為這場辯論畫下了句點。
