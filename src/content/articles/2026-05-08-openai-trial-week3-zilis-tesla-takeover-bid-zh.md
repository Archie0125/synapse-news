馬斯克針對 OpenAI 提起的 1,300 億美元訴訟，本週在奧克蘭聯邦法院迎來了迄今最具戲劇性的時刻：Shivon Zilis 登上證人席。Zilis 在本案中的身份極為特殊——她既是 OpenAI 前董事會成員（任期 2020 至 2023 年），同時也是馬斯克四個孩子的母親。這層錯綜複雜的個人關係，使她成為本次審判中外界最為關注的證人之一。

她在 5 月 6 日和 7 日兩天的證詞，揭示了 OpenAI 創立初期內部談判的混亂圖景、馬斯克對這家組織更宏大的野心，以及這場涉及數十億美元法律紛爭核心的人際糾葛。

## 特斯拉的收購圖謀

Zilis 作證中最重要的揭露：在 2017 至 2018 年間圍繞 OpenAI 公司架構的談判期間，馬斯克提議讓 OpenAI 實質性地與特斯拉合併，甚至被特斯拉吸收。為了說服 Sam Altman，馬斯克還提出給予 Altman 一個特斯拉董事會席位。

這個提議最終被拒絕。但其存在本身，已足以重構整個訴訟的核心敘事。馬斯克的法律團隊一直主張，他向 OpenAI 捐款數億美元，是基於 OpenAI 將維持非營利組織形態、服務全人類福祉的承諾——而 OpenAI 最終轉型為營利機構，是對這一承諾的背叛。然而，特斯拉董事會席位的提議，卻暗示了另一種解讀：馬斯克的真實目的，或許不僅是讓 OpenAI 維持非營利屬性，而是讓它成為自己個人帝國的一部分——融入他在特斯拉、SpaceX 等企業中苦心構建的版圖。

「那個時候，關於各種可能架構的爭論多到令人精疲力竭，」Zilis 作證說，她形容 OpenAI 領導層的內部討論充滿爭議。她表示，大家對公司架構問題的討論「就像反反覆覆嚼同一塊老牛肉」，足以說明「非營利」並非什麼既定教條，而是在那段歲月裡反復爭論、始終未有定論的問題。

## 邊任職邊挖角

Zilis 的證詞還帶出了對馬斯克一方頗為不利的文件證據。作為庭審展示物呈交的簡訊和電子郵件顯示，馬斯克在仍身為 OpenAI 董事會成員的同時，已積極著手從公司挖走關鍵人才。

一邊在 OpenAI 董事會任職，一邊試圖將其頂尖研究人員挖走——這一指控性質嚴重，在很大程度上動搖了馬斯克「被背叛的創始人、誓死捍衛 OpenAI 使命」的形象。倘若馬斯克真的以 OpenAI 治理者的身份監管這家公司，卻同時設法抽空其人才以打造競爭性 AI 能力，那麼他主張對 OpenAI 未來擁有道德話語權，便難以自圓其說。

## 保密協議與私人糾葛

Zilis 在作證時就其與馬斯克的個人關係只簡短提及，但仍有所披露。她確認自己與馬斯克就最初的「捐贈」簽署了一份保密協議——這是對精子捐贈安排的表述方式——協議包含「完全保密」條款，部分原因是為保護孩子免受馬斯克所面臨的安全威脅的波及。

這份保密協議與 OpenAI 訴訟本身並無直接關聯，但其存在，再度彰顯了 Zilis 作為證人所處的微妙困境。OpenAI 的辯護律師在交叉詢問中持續就她的雙重立場施壓；馬斯克的律師則試圖將她塑造為一位能夠印證馬斯克創業承諾版本的內部知情者。

最終，她的證詞對雙方來說都不是決定性的勝利：她承認公司架構問題曾被廣泛討論——這對馬斯克有利；她也承認馬斯克曾在任職期間謀劃挖角，以及試圖讓 OpenAI 歸入特斯拉旗下——這對 OpenAI 的辯護有利。

## Brockman 的 300 億美元持股

在 Zilis 出庭之前，OpenAI 總裁 Greg Brockman 完成了兩天的作證。在交叉詢問中，Brockman 披露，按 OpenAI 目前 8,520 億美元的估值計算，他個人持股價值約達 300 億美元。馬斯克的律師藉此主張，Brockman 在涉及 OpenAI 轉制為營利機構的一系列決策中，存在重大財務利益衝突。

Brockman 的證詞主要聚焦於 OpenAI 的創立過程、他與馬斯克在創業初期的關係，以及公司從非營利到限制性營利再到公益公司的演變邏輯。他堅持認為，架構調整是在資本競賽日益激烈的 AI 賽道上保持競爭力的必要之舉——而馬斯克的法律團隊則認為，這不能成為背棄創業使命的理由。

## 審判的現狀

經過兩週半的庭審，本案已留下了一份異常豐富的 OpenAI 早期歷史文件記錄——一段長期被神話化、卻鮮少在宣誓下接受審視的歲月。九人陪審團聽取了馬斯克本人（三天作證）、Brockman、Zilis，以及雙方多位輔助證人的陳述。

自庭審開始以來，預測市場對馬斯克的勝訴評估已明顯下滑。截至本週中，交易者對馬斯克勝訴的定價約為 40%，低於庭審前的水準。文件證據——包括特斯拉收購提議和挖角簡訊——顯然引發了陪審員和觀察人士的疑問：馬斯克聲稱的動機（捍衛 OpenAI 的非營利使命）是否真實反映了他在那段時期的實際意圖？

法官表示，在諮詢性陪審團作出建議後，她預計將於五月中旬作出裁決。訴訟的代價遠超馬斯克索求的 1,300 億美元損害賠償：若馬斯克勝訴，可能迫使 OpenAI 撤銷其轉制決定，嚴重衝擊這家公司今年早些時候一度觸及萬億美元估值的 IPO 計劃。

從庭審證詞中清晰浮現的一點是：OpenAI 的創業初期，遠非創業神話所描繪的那般目標一致、使命至上。那是一段充滿爭議、充滿即興發揮，以及充滿個人野心相互角力的歲月。這種複雜性對雙方的簡單敘事都構成了挑戰——而這，最終或許將決定法官更傾向於相信哪一方的陳述。
