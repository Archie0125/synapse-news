四大會計師事務所之一的 PricewaterhouseCoopers（PwC）決定押注 Claude，而且這一次完全不低調。PwC 與 Anthropic 週四宣布大幅擴大戰略聯盟：3 萬名 PwC 專業人員將接受 Claude 認證培訓，雙方將設立聯合卓越中心（Center of Excellence），並正式成立一個專注於財務轉型的全新業務部門。

這是 Anthropic 迄今為止對外披露規模最大的企業 AI 部署承諾，也可能標誌著各大 AI 實驗室競逐企業服務市場制高點的戰局，出現了第一個決定性轉折點。

## 不是試驗，而是營運模式的重建

PwC 與 Anthropic 的合作不是意向書，也不是概念驗證。它是一家全球最大專業服務公司對自身交付模式的結構性重組。合作從美國團隊啟動，逐步擴展至 PwC 全球數十萬員工，核心工具包括三款 Anthropic 產品：負責分析與知識整合的 Claude 主模型、用於軟體開發與自動化的 Claude Code，以及設計用於複雜多步驟專業工作流的多代理協作平台 Claude Cowork。

合作圍繞雙方定義的「三大高槓桿領域」展開：代理式技術建構、AI 原生交易撮合、企業職能再造。三個領域分別對應 PwC 的科技諮詢、交易服務、管理顧問三大業務線，目標一致——不是把 AI 疊加在舊流程上，而是以 AI 原生邏輯重建流程本身。

「我們不是把 Claude 貼在原有工作方式的表面上，」一名參與聯盟設計的 PwC 高層週四對記者表示，「我們是以 Claude 為核心，從頭建構新的營運模型。」

## 數字背後的意義

具體成果令人印象深刻。在已上線的生產部署中，PwC 客戶回報交付效率最高提升 70%。最具說服力的案例：一條原本需要十週的保險核保流程，已被壓縮至十天——五倍加速，讓過去因時程過長而不具經濟可行性的業務線重新成為可能。

在人才培育方面，PwC 計畫讓 3 萬名專業人員完成 Claude 認證課程，涵蓋 Claude 的通用能力與 Claude Code 在技術工作流自動化中的專項應用。PwC 全球擁有約 36.4 萬名員工，培訓 3 萬人約佔 8%，這已遠超一般試點計畫的規模，足以定性為核心人力發展策略。

## 全新財務業務部門上線

週四公告中最具體的結構性宣告，是 PwC 正式設立「財務業務部門」（Finance Business Group），專注於運用 Claude 協助客戶改造財務組織。這個新部門將 PwC 在資金管理、財務控制、FP&A 和審計方面的領域專業，與 Anthropic 的完整產品矩陣相結合——Claude 負責分析與綜合，Claude Cowork 負責自動化多代理財務工作流，Claude Code 則用於開發客製化的財務自動化工具。

財務職能之所以成為企業 AI 的優先進攻目標，邏輯並不難理解：資料密集、規則明確、長期以來自動化程度遠低於其潛力。那些多年來致力縮短結帳週期、加速對帳、提升預測準確度的財務長們，正越來越願意採納能自動化例行工作的代理式 AI。

PwC 的押注在於：率先將可信賴、以 Claude 為核心的財務轉型實踐帶入市場，能夠建立難以複製的競爭優勢。競爭對手中，德勤深度整合 Microsoft Copilot，畢馬威聯手 Google Cloud，但目前尚無任何事務所公布這等規模、以單一 AI 供應商產品為主軸的培訓計畫。

## Anthropic 的企業攻略

對 Anthropic 而言，PwC 合作案體現的是一套刻意有別於 OpenAI 的企業策略。當 ChatGPT 仍主導消費端心智時，Anthropic 已悄然建立起一份涵蓋 Salesforce、SAP、Accenture、BCG，以及現在的 PwC 的企業合作名單。模式一脈相承——與受信任的專業服務中介深度整合，讓 Claude 進入高合規要求、高風險決策環境，在那裡，可靠性與安全性和模型能力同等重要。

時機的選擇也值得關注。Anthropic 正準備公布 2026 年第二季財報，投資人最關心的問題是：AI 公司能否把 API 收入轉化為可持續的企業級經常性收入？讓 3 萬名顧問押注 Claude、讓 Claude Code 成為全球最具影響力顧問公司之一的預設開發環境，正是企業軟體分析師衡量黏性的標準指標。

此外，這份協議的時間節點也暗含競爭意味。Azure 雖已代管 Anthropic 的 Claude 模型，但微軟自身的 Copilot 產品線正與 Anthropic 直接競爭顧問工作流市場。PwC 的協議以 Anthropic 的原生產品——而非 Azure 代管版本——為核心，清楚傳達了一個方向。

## 這對整個產業意味著什麼

PwC 與 Anthropic 合作案更深遠的意涵，在於企業 AI 採用模式的結構性轉變。傳統路徑是：科技廠商直接向 IT 部門銷售，整合耗時 18 個月，採用率參差不齊。這套模式正在被 AI 原生顧問公司打破——後者將模型能力封裝進有主張的工作流，以高速度直接部署到客戶業務第一線。

PwC 的 3 萬名 Claude 認證顧問，未來將進駐數千家客戶組織。每一次部署都是參考案例，每一套工作流都是可複製的範本，每一個 70% 的效率改善都是 Anthropic 下一場商業對談的籌碼。在這個模型中，專業服務公司不只是客戶——它們是分發渠道。

如果這套模式在 PwC 奏效，德勤、安永、畢馬威、埃森哲、麥肯錫的類似公告只怕不會太遠。定義企業 AI 作業模式、掌握部署它的顧問層——這場競賽，已正式開打。
