2024年2月，當 OpenAI 首次展示 Sora 時，全球科技圈為之震撼。僅憑一段文字描述，就能生成高度逼真、彷彿違反物理定律的影片片段——那一刻，許多人以為媒體、娛樂與人類創意的未來已經提前到來。十八個月後，那扇窗口悄然關上。

2026年3月24日，OpenAI 低調宣布將終止 Sora 服務。消費者端的網頁與行動應用程式已於4月26日下線；供開發者使用的 API 將在2026年9月24日正式關閉。所有用戶資料屆時將被永久刪除，官方不承諾任何資料恢復期。

這個決定背後的數字，清楚說明了一切。

## 算術根本行不通

根據內部流出的資料與產業分析師的估算，Sora 每日推理運算成本高達約1500萬美元。大規模生成高解析度影片，所需的 GPU 算力遠比文字或圖像生成高出數十倍：Sora 每生成一分鐘影片，所消耗的算力約為一般 ChatGPT 對話的10至15倍。

對比如此驚人的日燒速率，Sora 的終身總營收估計僅約210萬美元——這個落差，坦白說已不像商業困境，更像是一場財務災難。用戶留存率同樣黯淡：每月200美元的個人 Pro 方案，訂閱30天後仍持續付費的用戶不到8%。下載量從高峰期的百萬次，崩跌至不足50萬名活躍用戶。

OpenAI 將 Sora 定位為面向專業創作者的高端工具，試圖以訂閱制實現商業化，但這個模式未能建立用戶持續付費的習慣迴路——想試試的人試了，願意每月掏出200美元的人卻根本沒有大量出現。

## 競爭對手的壓力

Sora 進入了一個進化速度超乎預期的市場。2025年中期，中國競品已大幅縮小技術差距。2026年初，Google 的 Veo 3.1 與快手旗下快科技研發的 Kling 2.5，在時序連貫性與物理真實感（避免物件違反物理法則的「幻覺」問題）上，被業界普遍評為已超越 Sora。

兩款競品的定價均明顯低於 Sora 每月200美元的 Pro 方案，加速了本可成為 Sora 核心客群的專業創作者出走。

更讓 OpenAI 措手不及的，是一筆原本可以改寫命運的合作告吹。Disney 曾洽談一項1.5億美元的內容授權協議，計畫利用 Sora 為旗下子公司生成視覺素材，但因技術品質疑慮與訓練資料的法律爭議，最終合約未能簽署。這個潛在的企業收入模式從未落地。

## 法律陰影揮之不去

除了財務壓力，Sora 始終籠罩在法律訴訟的陰影下。多起由藝術家、編劇與圖庫供應商提起的訴訟，指控 OpenAI 在未取得適當授權或給予創作者補償的情況下，以版權影片訓練 Sora。

雖然截至停服公告發布時，這些案件尚未有最終判決，但訴訟風險使企業合作難以推進。製片公司和內容機構對於在一個訓練資料來源仍有法律爭議的平台上建立工作流程，自然裹足不前。

## 釋出的算力流向何處

關閉 Sora，在某種意義上也是一場算力重新分配。OpenAI 正將這些釋出的運算資源，投入公司認為真正具有長期價值的領域。

其中最受矚目的是代號「Spud」的專案——一款桌面端超級應用程式，計畫在2026年9月配合 IPO 進程對外推出。Spud 被描述為以推理能力為核心的生產力環境，定位接近個人 AI 作業系統，而非單純的聊天機器人或內容生成工具。

算力也被導向公司日益壯大的機器人研究計畫，以及能夠跨軟體環境自主執行多步驟任務的智慧代理（agentic AI）系統。

一位 OpenAI 高層在流出的內部通訊中寫道：「純粹的生成已是解決了的問題；推理才是新的前沿。」這句話精準概括了公司的戰略轉向：生成令人信服的輸出——無論是圖像、影片還是程式碼——正在迅速被商品化。真正的持久優勢，OpenAI 押注在能夠推理、規劃、採取行動的系統上。

## 對 AI 影片產業意味著什麼

Sora 的停服，並非宣告 AI 影片生成的失敗，而是宣告 OpenAI 選擇的特定商業模式——為計算密集型產品收取高額消費者訂閱費——的失敗。

Google Veo 3.1 整合進 YouTube Studio 作為第一方功能，成本分攤在 Google 龐大的基礎設施與廣告業務上；Kling 2.5 憑藉快手的發行優勢，大幅降低獲客成本，在亞洲創意產業獲得廣泛的專業採用。這些平台的生存之道，在於嵌入更大的生態系，而非孤立運作。

對目前仍在使用 Sora API 的開發者與創作者而言，9月24日的關閉期限意味著遷移工作已刻不容緩。而在眾多替代方案中，最可能持續存活的，往往是那些嵌入更大生態系、擁有多元收入結構的平台。

## 一個更大的警示

在 AI 產品史上，Sora 將以一個里程碑被記住：它是業界首次清醒認識到，「令人印象深刻」與「經濟上可持續」是截然不同的兩件事的時刻。

那個在2024年震驚世界的展示，始終未能找到一個能夠支撐2026年算力成本與競爭格局的商業模式。

目前，Sora API 的倒數計時器還剩101天。此後，這個史上最受矚目的 AI 影片平台將徹底消失——留下的，是以運算帳單和空白訂閱記錄寫就的一堂商業課。
