OpenAI 期待已久的企業攻勢本週正式揭幕——ChatGPT Work 隆重登場。這個平台徹底重新定義了這家公司旗艦產品的能耐。7 月 9 日（週四）正式發表，並已開始向 Pro 和企業用戶推出，ChatGPT Work 與其說是聊天機器人的升級，不如說是 AI 融入專業工作流程方式的全面重塑。OpenAI 希望藉此終結 Anthropic 在企業市場日益穩固的地位。

## ChatGPT Work 到底能做什麼

簡而言之，ChatGPT Work 是一個自主執行環境。使用者只需提出一個目標——季度報告、競爭分析、可部署的網頁應用——代理便接手處理。它將目標拆解為較小的步驟，跨越連結的應用程式和工作流程蒐集資訊，並在必要時獨立作業數小時，最終交出完成品，而非需要人工修改的草稿。

產出內容相當具體：試算表、簡報、精修文件，以及可實際運行的網頁應用——整個過程不再需要來回反覆確認，這正是過去 AI 輔助工作的常態。OpenAI 官員形容，這是「給你靈感的工具」與「交付完成品的工具」之間的根本差異。

平台以 GPT-5.6 為核心，這是 OpenAI 本週隨 ChatGPT Work 一併推出的最新旗艦模型系列。其中針對代理式程式設計和電腦操作任務優化的 Sol 變體，在 Terminal-Bench 2.1（開發者終端機程式設計基準測試）上達到 88.8% 準確率，微幅超越 Anthropic Fable 5 的 88%。Altman 向 CNBC 表示，Sol 在代理式程式設計任務上的 Token 效率比前一版本提升 54%，對於大規模執行代理的企業部署而言，意味著顯著的成本節省。

## 整合產品線

ChatGPT Work 的推出同時重新整合了 OpenAI 日益龐雜的產品陣容。今年稍早以程式設計代理身份推出的獨立應用 Codex，將併入全新設計的 ChatGPT 桌面應用程式，把 Chat、Work 和 Codex 整合在統一介面下。值得注意的是，新桌面應用讓這三項功能在免費方案也可使用，大幅拓展了存取範圍。

工作區代理是另一個全新層次。企業版和商業版用戶現在可以建立自訂代理、測試、在組織內共用，並排程自動執行，包括整合進 Slack 的商業和企業工作區。代理可連結內部資料來源，獲取在特定公司環境中有效運作所需的業務脈絡，而非每次會話都從零開始。

## 直接瞄準 Claude Cowork

OpenAI 發布聲明中的競爭定位出奇地直白。多位高層主管明確表示，ChatGPT Work 是對 Anthropic Claude Cowork 的直接回應。Cowork 是 Anthropic 今年 1 月推出的代理式工作空間產品，過去六個月已在金融服務、法律和專業服務公司積累了可觀的企業客戶群——這些公司的日常工作正是複雜的多步驟文件工作流程。

Anthropic 搶先起跑，而且確實發揮了作用。這段先機讓 Anthropic 在企業採購談判中擁有相當大的籌碼，因為已將內部流程建構在 Cowork 代理架構上的 IT 部門，面臨不低的轉換成本。OpenAI 官員表示，ChatGPT Work 將「比競爭產品更便宜、覆蓋面更廣」——這是對 Cowork 企業定價策略，以及上週以高價登場的 xAI Grok 4.5 的雙重暗諷。

ChatGPT Work 的定價結構尚未完整披露，但 OpenAI 表示將將其納入現有 ChatGPT 訂閱方案，而非作為獨立產品銷售——對已訂閱 Pro 或商業方案的用戶而言，等同以更低的增量成本獲得代理能力。

## 更大範圍的平台整合

ChatGPT Work 是 AI 助理市場正在發生的更大整合的一部分。隨著前沿模型能力趨於收斂——GPT-5.6 Sol、Claude Fable 5 和 Grok 4.5 在多數標準基準測試上的差距已縮小至個位數百分點——競爭戰場已轉向工作流程整合、大規模代理可靠性，以及企業發行渠道。

OpenAI 的賭注是：將代理能力捆綁進全球最具知名度的 AI 產品，足以克服 Anthropic 的先發優勢。Anthropic 的反向賭注則是：企業 IT 團隊一旦深度整合 Cowork 的代理架構，就會形成高度黏著性。未來兩季的企業交易流向，將決定哪個論點更接近現實。

兩家公司據報都在籌備潛在的首次公開發行，企業營收數字因而受到機構投資人日益嚴密的審視。ChatGPT Work 為 OpenAI 提供了一個清晰的答案：如何在下一個規模的 AI 採用中實現商業化——不是靠授權智慧聊天機器人的使用權，而是靠已完成的自主工作收費。

## 開發者需要關注什麼

對於在 OpenAI 平台上開發的工程師而言，Work 的推出帶來了工作區代理 API，簡化了過去需要透過 Assistants API 進行複雜編排才能實現的許多功能，新端點讓代理的建立、排程和企業環境整合更加便利。

Codex 併入統一 ChatGPT 桌面應用的決定，也傳遞出一個訊號：OpenAI 現在將程式碼生成視為通用工作平台中的一項能力，而非獨立的專業工具。過去仰賴獨立 Codex 應用建立工作流程的開發者，應密切注意過渡時程，因為 OpenAI 計畫在未來幾週內逐步淘汰獨立版本。

本週發布傳遞的核心訊息再清楚不過：OpenAI 將企業市場視為 AI 採用下一階段的核心戰場，而且不再甘心於在模型基準測試上競爭的同時，將這塊市場拱手相讓給 Anthropic。
